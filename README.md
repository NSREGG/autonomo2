Piedra, Papel o Tijeras — Proyecto de Lógica de Programación

Objetivo del programa

Este proyecto tiene como finalidad aplicar los conceptos fundamentales de la lógica de programación mediante la creación de un juego interactivo en Python. El jugador compite contra la computadora eligiendo entre piedra, papel o tijeras, y el programa determina el resultado usando estructuras condicionales, funciones y bucles.

Principales funcionalidades del código

Entrada del jugador: El usuario ingresa su elección mediante teclado.

Elección aleatoria de la computadora: Se utiliza el módulo random para simular la decisión del oponente.

Comparación de elecciones: Se evalúan las reglas del juego con condicionales if, elif, else.

Determinación del resultado: El programa informa si el jugador ganó, perdió o empató.

Repetición del juego: Se implementa un bucle while que permite jugar múltiples rondas hasta que el usuario decida salir.

Organización modular: Se utiliza una función determinar_ganador() para encapsular la lógica de comparación.
